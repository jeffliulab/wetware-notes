# 湿件计算

> *Wetware Computing*

湿件计算把活体神经元网络当作计算基质——体外培养的神经元学习、适应并执行计算。这是"用活体算"的一环，把神经科学、认知科学与生物基质收束为一种全新的类脑计算范式。

## 章节规划

1. **导论** — 湿件计算的定义、动机与全景。
2. **体外神经元网络** — 在培养皿中生长可计算的神经网络。
3. **电刺激与多电极记录** — 读写活体神经网络的接口技术。
4. **体外学习与可塑性** — 培养神经元如何学习——DishBrain 范式。
5. **生物混合与闭环系统** — 把活体网络嵌入感知-行动回路。
6. **物理与储备池计算范式** — 把生物动力学当作计算资源的范式。
7. **伦理与前沿** — 湿件智能的伦理边界与未来方向。
